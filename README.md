# Aloa's Restaurant & Beach Bar

A one-page bilingual (Shqip / English) website for **Aloa's Restaurant & Beach Bar** in Qeparo, on the Albanian Riviera.

🌊 *Aty ku malet takohen me detin Jon — Where the mountains meet the Ionian Sea.*

## What's here

- **`index.html`** — the complete site (HTML + CSS + JS, no build step, no dependencies).
- **`assets/img/`** — logo and photos.
- **`assets/video/`** — background videos, re-encoded for desktop (`hero-desktop.mp4`, 1280×720) and mobile (`hero-mobile.mp4`, 720×1280).

## Features

- 🇦🇱 / 🇬🇧 instant language toggle (Albanian default)
- Responsive hero video — loads only the clip matching the device orientation
- Menu with prices in Lekë and approximate Euro
- Full-menu chalkboard in an in-page lightbox
- Google Maps location, click-to-call phone, Instagram link
- Fully responsive, no frameworks

## Run locally

Just open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Deploy (GitHub Pages)

Settings → Pages → Build from branch → `main` / root. The site is static and works as-is.
